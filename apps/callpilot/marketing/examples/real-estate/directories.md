## Claiming Checklist

For each directory:

- [ ] Create/claim listing
- [ ] Add consistent NAP (Name, Address, Phone)
- [ ] Upload high-quality logo
- [ ] Add business description with keywords
- [ ] Upload exterior/interior photos
- [ ] Set accurate business hours
- [ ] Add service areas
- [ ] Link to website
- [ ] Request reviews from past clients
