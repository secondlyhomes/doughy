## Listing Optimization Checklist

For each directory:

- [ ] Create complete profile
- [ ] Add detailed product description
- [ ] Upload high-quality screenshots
- [ ] List all key features
- [ ] Add pricing information
- [ ] Specify integrations
- [ ] Add comparison content
- [ ] Request reviews from customers
- [ ] Respond to all reviews
- [ ] Update quarterly with new features

## Review Acquisition Strategy

### G2 Reviews

1. Create G2 review campaign link
2. Add to post-purchase email flow
3. In-app prompt after key milestones
4. Personal outreach to power users

### Product Hunt

1. Plan launch carefully (timing matters)
2. Build launch community beforehand
3. Prepare assets (logo, description, video)
4. Engage on launch day

### General Tips

- Don't incentivize reviews (violates TOS)
- Ask happy customers specifically
- Make leaving review easy (direct links)
- Respond to all reviews professionally
