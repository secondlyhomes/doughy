# Tutorials

Step-by-step interactive tutorials to get you productive quickly with the Mobile App Blueprint.

## Getting Started

Start here if you're new to the blueprint:

### [Tutorial 1: Getting Started](./01-getting-started.md)
**Duration:** 45-60 minutes

Set up your development environment and run your first app.

**You'll learn:**
- Installing prerequisites (Node.js, Expo, EAS CLI)
- Cloning and configuring the project
- Setting up Supabase
- Running on simulator/device
- Making your first change
- Understanding the project structure

**Prerequisites:** Basic JavaScript knowledge

---

## Building Features

Once you have the basics, build complete features:

### [Tutorial 2: Building Your First Feature](./02-first-feature.md)
**Duration:** 90-120 minutes

Build a complete task management feature from scratch.

**You'll learn:**
- Feature planning and structure
- Database schema design with RLS
- Service layer patterns
- Component composition
- State management
- Real-time subscriptions
- Testing strategies

**Prerequisites:** Complete Tutorial 1

---

## Authentication

Add user authentication to your app:

### [Tutorial 3: Adding Authentication](./03-adding-authentication.md)
**Duration:** 90-120 minutes

Implement complete authentication with Supabase Auth.

**You'll learn:**
- Email/password authentication
- Social OAuth (Google, Apple, GitHub)
- Session management
- Protected routes
- Profile management
- Password reset flows
- Email verification

**Prerequisites:** Complete Tutorial 1

---

## Advanced Database

Master database techniques:

### [Tutorial 4: Advanced Database Integration](./04-database-integration.md)
**Duration:** 120-150 minutes

Learn advanced Supabase database features.

**You'll learn:**
- Relationships and joins
- Complex queries and filtering
- Transactions with database functions
- Real-time subscriptions
- Performance optimization
- Full-text search
- Migration strategies

**Prerequisites:** Complete Tutorials 1-2

---

## Deployment

Ship your app to production:

### [Tutorial 5: Deployment](./05-deployment.md)
**Duration:** 180-240 minutes

Build, deploy, and publish to App Store and Play Store.

**You'll learn:**
- Pre-deployment checklist
- Building for iOS and Android
- App Store submission
- Play Store submission
- OTA updates with EAS Update
- CI/CD with GitHub Actions
- Production monitoring

**Prerequisites:** Complete Tutorials 1-3

---

## Learning Path

**Recommended order:**

```
1. Getting Started (Required)
   ↓
2. First Feature (Recommended)
   ↓
3. Authentication (Recommended)
   ↓
4. Database Integration (Optional)
   ↓
5. Deployment (When ready to ship)
```

**Time commitment:**
- **Minimum path** (1, 2, 3): 4-6 hours
- **Full path** (all tutorials): 10-14 hours
